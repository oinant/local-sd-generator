# CLI Commands

- sdgen generate => run a generation from a prompt file
- sdgen list => list all templates/prompt files
- sdgen validate => validate a prompt/template file
- sdgen init => init sdgen context within a folder
- sdgen api => introspect a1111
- sdgen help => show help
- sdgen start => start the env : start a1111 (depending on os/wsl... ), start the backend and the frontend
- sdgen stop => stop the env
- sdgen webui => status of the web ui (back + front)
- sdgen webui start => start the backend and the frontend
- sdgen webui restart => restart the backend and the frontend
- sdgen webui stop => stop the backend and the frontend
