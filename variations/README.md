# Variations Directory

Ce répertoire contient les fichiers de variations partagés entre le CLI et le backend web.

## Format des fichiers

Les fichiers de variations suivent le format :
```
# Commentaires supportés
1→angry
2→happy,smiling
surprised→very surprised
just_a_value
```

## Fichiers d'exemple

Créez vos fichiers de variations ici :
- `facial_expressions.txt`
- `angles.txt`
- `lighting.txt`
- `styles.txt`
- etc.
